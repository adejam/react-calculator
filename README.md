# REACT-CALCULATOR

> React-calculator is a calculator app designed using HTML,CSS and Javascript and React.

![screenshot](./app_screenshot.png)

The project demonstrates the use of React to create a calculator app.

## Features

- Basic binary operations
- Calculations with big numbers

## Technology Used

- HTML

  - Semantic HTML

- CSS

  - CSS Flexbox

- [React](https://reactjs.org/)

- Javascript

  - Arrow function.
  - Destructuring assignment.
  - Object literal
  - Template literal
  - Ternary Operators

### Development (Running locally)

- Clone the project

```bash
git clone https://github.com/adejam/react-calculator.git

```

- Install Dependencies

```bash
npm install
```

To run StyleLint by itself, you may run the lint task:

```bash
npm run lint:check
```

Or to automatically fix issues found (where possible):

```bash
npm run lint
```

You can also check against Prettier:

```bash
npm run format:check
```

and to have it actually fix (to the best of its ability) any format issues, run:

```bash
npm run format
```

You can also check against HTML Validator:

```bash
npm run html-validator
```
